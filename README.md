# RecallX
RecallX is a smart memory recall and productivity system built using Django APIs to help users store, manage, and retrieve important information efficiently.

📌 Features
🧠 Store important information easily
🔍 Quick search and retrieval system
⚡ Fast and efficient backend using Django
📊 Organized data management
🌐 User-friendly interface

🛠️ Tech Stack
Backend: Django
Frontend: HTML, CSS, JavaScript
Database: SQLite
API: Django REST Framework


📂 Project Structure
backend/ → Core backend logic
django_app/ → Main Django project
frontend/ → UI part
Media/ → Uploaded files/images
requirements.txt → Dependencies

⚙️ Installation & Setup
git clone https://github.com/your-username/RecallX.git
cd RecallX
pip install -r requirements.txt
python manage.py runserver

🎯 Purpose:
This project is developed to improve productivity by helping users efficiently manage and recall important data.
